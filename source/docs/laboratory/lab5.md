## 📋 Описание задания

В данной лабораторной работе рассматривается задача регрессии, то есть задача предсказания непрерывного числового значения на основе набора признаков.

В ходе работы необходимо:

- загрузить и исследовать датасет;
- выполнить предобработку данных;
- разделить данные на обучающую и тестовую выборки;
- обучить несколько моделей регрессии;
- оценить качество моделей с помощью метрик MSE, MAE и RMSE;
- сравнить результаты моделей;
- исследовать параметры модели Random Forest и попытаться улучшить качество предсказания.

---

## 💡 Решение

Решение задачи было выполнено поэтапно.

### 1. Загрузка и анализ данных

Сначала был загружен датасет и выполнен его первичный анализ. Были изучены типы признаков, наличие пропусков, а также основные статистические характеристики данных.

### 2. Предобработка данных

На этапе предобработки были выполнены следующие действия:

- удаление или заполнение пропущенных значений;
- обработка категориальных признаков;
- выделение целевой переменной;
- формирование итогового набора признаков для обучения модели.

Если в данных присутствовали категориальные признаки, для них применялось кодирование.

### 3. Разделение на обучающую и тестовую выборки

После подготовки данных выборка была разделена на две части:

- **train** — для обучения моделей;
- **test** — для проверки качества.

Это позволяет объективно оценить, насколько хорошо модель работает на новых данных.

### 4. Обучение моделей регрессии

Для решения задачи были использованы несколько моделей регрессии, например:

- Linear Regression;
- Random Forest Regressor.

Также могли быть рассмотрены и другие модели, если это предусмотрено в ноутбуке.

### 5. Оценка качества моделей

Для оценки качества работы моделей использовались следующие метрики:

- **MSE** (Mean Squared Error) — средняя квадратичная ошибка;
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка;
- **RMSE** (Root Mean Squared Error) — квадратный корень из средней квадратичной ошибки.

Чем меньше значения этих метрик, тем лучше модель решает задачу регрессии.

### 6. Сравнение результатов

После обучения моделей были получены и сравнены значения метрик. Это позволило определить, какой алгоритм лучше подходит для данной задачи.

### 7. Дополнительное исследование

Дополнительно были исследованы:

- важность признаков;
- влияние выбора параметров модели Random Forest;
- изменение качества модели после настройки гиперпараметров.
