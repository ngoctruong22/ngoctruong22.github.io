## Выбор темы оформления

Этот сайт использует тему **Material for MkDocs**. Вот почему я выбрал именно её:

- *Современно и профессионально:* Material Design выглядит красиво и чисто, отлично подходит для портфолио и технической документации.
- *Много полезных функций:* Есть поиск, подсветка кода, тёмная и светлая темы, удобная навигация.
- *Большое сообщество:* Эту тему используют многие разработчики, есть подробная документация и хорошая поддержка.
